Football Competition Analysis Tool

Language:

Enter match odds to simulate group outcomes and analyze probabilities. Lambdas are derived using an iterative xG calculation based on O/U and Home/Away odds.

Use Elo mode when you only have team ratings by group. Expected goals are generated from Elo differences.

Elo CSV supported format

Accepted delimiters: comma, semicolon, or tab.

Required columns: GROUP,TEAM,ELO (header optional).

Hybrid mode uses your provided odds matches, then fills missing group fixtures from Elo.

Hybrid input structure (proposed)

1) Match Data (Odds Input): include only known matches (e.g., Round 1), using the existing odds row format.

2) Elo Data: include all teams per group using GROUP,TEAM,ELO, so missing pairings can be generated.

Example rows:

GROUP,TEAM,ELO
A,Germany,1942
A,Scotland,1730
B,Spain,2036
